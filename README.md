# Book Tracker

**Автор:** Аделя Петрова

## Описание программы
Программа «Book Tracker» — это GUI-приложение для управления списком прочитанных книг. Позволяет добавлять книги, фильтровать их по жанру и количеству страниц, сохранять и загружать данные в формате JSON.
